# rpg127_feature_pack_safe

rpg126の32x32敵スプライト/OAM制限対応版を基準に、検討中機能を小さく追加した版です。

## 追加機能
- 通常攻撃による武器熟練成長
- 戦闘後のアクティブメンバー全員成長
- 戦意・集中・疲労・冒険勘のステータス表示

## 実装方針
Bank 0が厳しいため、重い処理は `party_runtime.c` の bank 7 側に寄せています。
`main.c` への追加は通常攻撃時の通知と、戦闘勝利後の成長呼び出しに限定しています。

## ビルド
通常通り:

```sh
chmod +x build.sh
./build.sh
```

`romusage` で Bank 0 overflow が出ないか確認してください。
