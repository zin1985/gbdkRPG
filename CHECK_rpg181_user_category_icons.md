# rpg181_user_category_icons

## 変更内容

- ユーザー提供の横64x縦8の8ドットアイコンをGB 2bpp BGタイル8枚へ変換。
- BGタイル244〜251へロードする `ui_icons.c/.h` を追加。
- 左から順に、道具、剣、杖、道具武器、弓矢、鎧、服、アクセサリーとして使用。
- もちもの一覧のアイテム名左にカテゴリ別アイコンを表示。
- そうび画面の装備中アイテム、装備候補ポップアップ、武器種表示にアイコンを表示。
- 店の商品一覧にもカテゴリ別アイコンを表示。

## Bank 0方針

- `main.c` は未変更。
- アイコンタイルデータと描画関数は `#pragma bank 8` の `ui_icons.c` に配置。
- JPフォントキャッシュ(128〜223)、UI枠(224〜232)、ミニマップ(240〜243)と競合しないよう、244〜251を使用。

## 注意

- GBDK `lcc` がない環境では実ビルド未確認。
- ビルド時は Bank 7/8 と Bank 0 overflow、`Write from one bank spans into the next` を確認してください。
