# rpg187 使用可能文字一覧 / フォント引継ぎメモ

## 概要

この版は `rpg186_official_misaki_rpg_kanji` をベースに、現行 `misakiUTF16FontData.h` の `ftable` から使用可能文字を整理したドキュメント追加版です。  
コード本体・フォント本体・`main.c` は変更していません。

- 現行フォント総数: 1542 文字
- 使用可能漢字: 1095 文字
  - もともとの教育版ベース漢字: 1007 文字
  - rpg185/rpg186で追加したRPG頻出漢字: 88 文字
- フォント本体: Bank 6
- Bank 0増加: なし

## 注意

- ギリシャ文字、キリル文字、全角英数字、半角カナ、`㈱` などの互換記号・単位系は削除済みです。
- 現行テーブルには、記号枠として `℃№℡Å`、ローマ数字、丸数字、数学記号などがまだ残っています。不要なら次回さらに削減候補です。
- 半角ASCII英数字は現行テーブルにも残っていますが、将来的にはGBDK標準フォントへ寄せる余地があります。

## 4. 実装・運用メモ

- アイテム名・装備名・技名では、この一覧にない漢字を使うと文字化けします。
- `賢 / 杖 / 剣 / 鎧 / 魔 / 飾 / 雷 / 闇` は rpg186 で公式美咲ゴシック字形に差し替え済みです。
- 新しいアイテム名を追加する時は、まずこのファイルで使用文字を確認してください。
- 文字を追加する場合は、Bank 6の容量と `ftable` / `fdata` の整合性を確認してください。
- Bank 0保護方針は継続。フォント追加・差し替えは `misakiUTF16FontData.h` と Bank 6側に閉じるのが安全です。
