# CHECK rpg092_ui_fix

## 目的
- 1ターン目のコマンド選択時だけ起きる戦闘UIズレを抑える
- ゆうしゃアイコンが消える症状を抑える
- メッセージ枠を復活させる
- ダメージ量を数値で表示する

## 今回の実装
- 初回プレイヤーターン突入前に `battle_prepare_player_turn_ui()` で戦闘UIを再整列
- メッセージ表示領域を枠付きで再描画
- 戦闘メッセージを1行用に圧縮表示
- 攻撃/とくぎ/敵攻撃時に数値ダメージ表示を追加

- 攻撃時・敵攻撃時にダメージ値が表示される
- 戦闘後にフィールドへ正常復帰する
