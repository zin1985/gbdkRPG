# rpg130_world_dungeon_feature_pack

## 概要

町・港町・洞窟・古代遺跡を追加し、世界を広げた版です。
敵3体32x32のBG表示方式は維持しています。

## 追加エリア入口

- 町: フィールド `x=14, y=2`
- 港町: フィールド `x=14, y=14`
- 洞窟: フィールド `x=1, y=14`
- 古代遺跡: フィールド `x=1, y=2`

## 追加機能

- 港町マップ
- 洞窟ダンジョンマップ
- 古代遺跡マップ
- ダンジョン用BGマップチップ
- 宝箱取得/空箱
- 罠床小ダメージ
- 危険エリアのランダムエンカウント率変更
- 古代遺跡メッセージ
- クエスト状態の簡易進行

## Bank 0方針

大量データはbanked側へ寄せています。main.cの追加はエリア遷移・イベント判定に絞っています。
ただし正確な判定は、ユーザー環境で `./build.sh` 実行後の `romusage` を確認してください。
