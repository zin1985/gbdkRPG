# rpg196_item_equip_pages_town_layout

## 対応内容
- 戦闘中どうぐを、通常どうぐ欄と同じ2列・ページ式UIで運用。
  - 下端で次ページ、上端で前ページへ移動。
  - 対象選択UIを追加し、誰に使うか選択してから使用。
- 移動中のどうぐ使用対象UIで、仲間名を `party_get_active_name()` から表示するよう補正。
- 装備候補UIを改善。
  - カーソル移動時に毎回ポップアップ全再描画しない。
  - 同一ページ内ではカーソルだけ差分更新。
  - 下端で次ページ、上端で前ページに移動し、末尾/先頭での循環は抑制。
- アイテム効果量をランダム化。
  - 回復アイテムは回復量をランダム化。
  - パラメーター増加アイテムは +1〜+3 の範囲でランダム化。
  - 使用後に数値をアナウンスするため、直近効果量を `party_runtime` に保持。
- 街レイアウトをユーザー指定配列ベースに更新。
  - オブジェクトがある場所は基本 collision=1。
  - ただし出口 `3` は通行できるよう collision=0 維持。
  - 上部の `7` を店扱いにし、前/左右ブロックで会話イベント発火。
  - 道具屋付近に道具アイコン、武器屋付近に剣アイコンを配置。
  - 中央付近の `7` は村人会話イベントに変更。
- 町専用オーバーレイタイル追加。
  - `kind 4` = 道具アイコン看板
  - `kind 5` = 剣アイコン看板
  - `kind 7` = ツボ/村人系オブジェクト表示

## 注意
- `MAP_GFX_TILE_COUNT` は増やしていません。
- 町表示時のみ既存タイルスロットを上書きする方式です。
- この環境ではGBDKの実ビルドは未実行です。
- ビルド後は romusage の Bank7 overflow と Bank5/11 の増加を確認してください。
