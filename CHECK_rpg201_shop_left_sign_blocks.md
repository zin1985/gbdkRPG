# rpg201_shop_left_sign_blocks

## 対応内容
- 店員の左ブロックに、店種別のBG合成ブロックを配置。
  - 左の店: ブロック + 「宿」文字
  - 真ん中の店: ブロック + 道具アイコン
  - 右の店: ブロック + 剣アイコン
- 町マップの上部店レイアウトを修正。
  - 店員位置は `(4,2) / (8,2) / (12,2)` のまま。
  - 左ブロック `(3,2) / (7,2) / (11,2)` に看板を配置。
- 新規 object kind:
  - `8` = 宿アイコン合成ブロック
  - `4` = 道具アイコン合成ブロック
  - `5` = 剣アイコン合成ブロック
- `MAP_GFX_TILE_COUNT` は増やしていません。
- 町表示時だけ、既存の未使用寄りタイルスロットを上書きします。

## 容量メモ
- 宿看板1種類の追加で 16x16 = 4タイル = 約64 bytes。
- 道具/剣看板は既存の合成看板を利用。
- BG合成なのでOAMを消費せず、スプライト制限にも影響しません。

ビルド後は `tools/prebank_static_check.py --strict` と romusage を確認してください。
