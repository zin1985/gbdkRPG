# CHECK rpg101_field_player_sheet_replace

## 目的
- ユーザー提供の 12x4 シートからフィールド用 16x16 スプライトを生成する
- ゆうしゃのフィールドスプライトを差し替える
- まほう / そうりょ / 4人目の仲間の field sprite データも追加する

## VBA確認
- フィールド上のプレイヤーが新しい ゆうしゃ スプライトになっている
- 下 / 上 / 左 / 右 の向きが崩れていない
- 歩行2フレームがそれぞれ反映されている
- 背景白が四角く残らず透明になっている
- 白 / 黒 / 中間色 の3色で表示されている
- ビルド時に player_tiles 差し替えでリンクエラーが出ない

## 補足
- 今回のメイン反映対象は `player_tiles`（ゆうしゃ）
- `mage_field_tiles` / `priest_field_tiles` / `fourth_party_field_tiles` は今後の仲間表示・切替用の下準備
