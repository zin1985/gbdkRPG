# rpg179_menu_shop_skill_item_ui_fix

## 変更内容

- メニュー下部はボタン説明を消し、所持金表示のみへ変更。
- 移動中オーバーレイは、パラメータとミニマップを横並びにする横長ウィンドウへ変更。
- 町の店配置を「上段壁・中央店員・前/横で会話判定」の形へ調整。
- 店イベント判定を店員の前と横のブロックにも追加。
- 戦闘中特技ウィンドウを4行表示に変更し、4つ目の特技が見えるよう修正。
- 移動中特技の流れを「とくぎ → 誰の → 特技一覧 → 対象」に変更。
- パラメータ下部の漢字ラベルをひらがな中心へ変更。
- 武器種表示もひらがな化。
- パラメータ下部が枠に被らないよう、空白行以降を1段上へ詰めた。
- はやあしリングをそうりょへ初期装備。
- はやあしリング装備時、移動速度を2倍化。
- アイテム名左に種類アイコン（ど/ぶ/よ/か）を表示。
- 戦闘中の「にげる」枠を「どうぐ」に変更し、戦闘中アイテム自動使用を追加。

## Bank 0方針

- `main.c` は修正前 96553 bytes → 修正後 96050 bytes。
- 戦闘中アイテム選択の本体処理は `inventory.c` 側へ寄せ、Bank 0増加を避けた。
- マップ・店・特技・パラメータUIの大半は既存BANKEDファイル側で修正。
