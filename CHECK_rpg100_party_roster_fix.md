# CHECK rpg100_party_roster_fix

## 目的
- ターン経過で仲間名が化ける問題を抑制する
- ゆうしゃとまほうの戦闘アイコンを交換する
- そうりょ、まほうを実データとして実装する
- 予備仲間3名を追加し、将来の入れ替えに備える

## 修正内容
- `jpfont.h`
  - `JP_CACHE_SIZE` を 48 → 96 に拡張
  - `JP_FRAME_BASE` を 176 → 224 に移動
  - 戦闘中の文字種類増加による glyph cache 再利用で、上部名表示が化ける症状を抑える

- `main.c`
  - `PartyMember party_roster[6]` を追加
  - active: ゆうしゃ / そうりょ / まほう
  - reserve: せんし / かりうど / ぶとうか
  - `party_swap_active_with_reserve()` を追加
  - 敵攻撃対象を生存中のアクティブ仲間からランダム選択
  - パーティステータス欄を party_roster から描画
  - HP/MP更新時にパーティ欄を再描画して名前化けを修復
  - ゆうしゃとまほうのアイコン tile offset を交換

## 今回まだ未実装
- 入れ替え用メニューUI
- そうりょ/まほうの個別コマンド入力
- 予備仲間の戦闘中交代操作
- 仲間別成長処理

## VBA確認
- ターン経過後も `ゆうしゃ / そうりょ / まほう` の名前が化けない
- ゆうしゃとまほうのアイコンが交換されている
- そうりょ、まほうのHP/MPが固定ダミーではなく party_roster から出ている
- 敵攻撃でゆうしゃ以外のHPも減ることがある
- 全員HP0で敗北になる
- 既存戦闘UI位置が崩れていない
