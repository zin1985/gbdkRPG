# rpg231_sprite_reward_member_skill_fix

## 修正内容

- 移動スプライト欠け対策
  - `SAFE_DUPLICATE_FIRST_ACTOR_FRAME` を有効化し、移動方向タイルが欠ける場合でも全フレームへ安定した1フレームを複製ロードするように変更。
- 閃いた名前を表示
  - 閃き時に「ひらめいた!」の後、特技/魔法名を表示。
- お金取得表示バグ修正
  - 戦闘後のお金取得で「ダメージ」表記にならないよう、専用メッセージ `おかね\nNNGET` 形式へ変更。
- 魔法熟練度ステータス表示
  - ステータス2ページ目の熟練度欄に `まほう` 熟練度を追加。
- 特技を仲間ごと個別管理
  - `main.c` の共通 `player_skills` を廃止。
  - `party_runtime.c` の各仲間に `learned_skills[]` を持たせ、戦闘中の特技/魔法一覧は行動中の仲間ごとに生成。
- 特技追加
  - 追加特技/魔法IDを `SKILL_MAX 71` まで拡張。
  - 武器系上位技、魔法上位技を追加。

## 確認ポイント

- `./build.sh` で危険警告が出ないこと。
- 移動中にプレイヤーのスプライトが欠けないこと。
- 戦闘後のお金表示が「ダメージ」にならないこと。
- 閃き時に技名/魔法名が表示されること。
- 仲間ごとに特技/魔法の一覧が分かれること。
- ステータス熟練度ページに魔法熟練度が出ること。
