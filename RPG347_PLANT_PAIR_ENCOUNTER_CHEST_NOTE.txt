RPG347 PLANT PAIR ENCOUNTER CHEST NOTE

User request:

- Keep the expanded enemy-name window behavior.

- encounter 9 is now プラントゲイズ + プラントアイ.

- Added a paired plant overlay draw path: when two plant-type enemies are present, the 8 safe enemy overlay OBJ slots are split as 4 OBJ for each enemy.
- Existing enemy-name window is kept at width 13, overlapping the command window by four BG blocks (x=9..12).
