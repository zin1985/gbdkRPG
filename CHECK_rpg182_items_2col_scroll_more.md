# rpg182_items_2col_scroll_more

## 変更内容

- もちもの一覧を2列表示へ変更。
  - 左右2列、上下左右カーソル移動。
  - 画面内に収まらない場合はカーソル位置に合わせて下方向へスクロール。
  - 各項目は `カーソル + アイコン + 短いアイテム名 + 個数` の詰め表示。
- `INVENTORY_ITEM_MAX` を 64 から 128 へ拡張。
- 使用アイテムを追加。
  - HP回復、MP回復、全快、状態回復、能力上昇、蘇生系など。
- 装備アイテムを追加。
  - 剣、杖、弓、手甲/道具武器、服、鎧、アクセサリーを追加。
- 追加アイテムにも、rpg181で導入した8ドットカテゴリ別アイコンを反映。
- 装備候補ポップアップを候補数増加に合わせてスクロール表示化。
- セーブデータ形式変更に伴い `SAVE_VERSION` を 3 に更新。

## Bank 0方針

- `main.c` は変更していません。
- 追加した一覧・名前・アイコン判定・装備定義は banked 側に配置しています。
- Bank 0を増やさない方針を維持しています。

## 注意点

- `INVENTORY_ITEM_MAX` 拡張により、既存セーブとは互換性がありません。
- 実ビルドはこの環境にGBDK `lcc` が無いため未実行です。
- ビルド時は特に Bank 7 / Bank 8 の `Write from one bank spans into the next` を確認してください。
