# CHECK rpg089_skill_slots

## 目的

rpg088_mp_skill を基準に、戦闘UIを増やさず、内部だけを複数技対応へ寄せる。

## 追加確認

- [ ] 起動して白画面にならない
- [ ] フィールド表示が崩れていない
- [ ] ランダムエンカウントする
- [ ] 戦闘画面のウィンドウ位置が rpg088 から変わっていない
- [ ] 「とくぎ」で MP が 2 減る
- [ ] 「かいふく」で MP が 4 減る
- [ ] HP満タン時の「かいふく」で MP を消費しない
- [ ] MP不足時にプレイヤー選択へ戻る
- [ ] 「とくぎ」「かいふく」が内部 skill slot 経由でも従来通り動く
- [ ] 戦闘勝利後に成長処理が動く
- [ ] 戦闘後にフィールドへ戻れる

## 変更しないこと

- [ ] 技一覧UIを追加していない
- [ ] 対象選択UIを追加していない
- [ ] 新規BGタイルを追加していない
