# CHECK rpg083_music

## 目的

フィールド・町・戦闘でBGMが切り替わることを確認する。

## ビルド確認

- [ ] `build.sh` に `audio.c` が含まれている
- [ ] `build.bat` に `audio.c` が含まれている
- [ ] `Makefile` に `audio.c` が含まれている
- [ ] `rpg083_music.gb` が出力される
- [ ] dangerous linker warning が出ていない
- [ ] bank overflow / spans warning が出ていない

## 音楽確認

- [ ] 起動後、フィールドBGMが鳴る
- [ ] フィールド移動中にBGMが継続する
- [ ] 町に入ると町BGMへ切り替わる
- [ ] 町から出るとフィールドBGMへ戻る
- [ ] ランダムエンカウント時に戦闘BGMへ切り替わる
- [ ] 戦闘勝利後にフィールドBGMへ戻る
- [ ] 戦闘敗北/逃走後にフィールドBGMへ戻る
- [ ] 会話中にBGMが完全停止しない
- [ ] 戦闘メッセージのA待ち中にBGMが完全停止しない
- [ ] メニュー中にBGMが完全停止しない

## 既存機能確認

- [ ] 起動して白画面にならない
- [ ] フィールドBGが表示される
- [ ] プレイヤーが表示される
- [ ] 移動できる
- [ ] NPC会話がフリーズしない
- [ ] ランダムエンカウントする
- [ ] 戦闘開始エフェクトが出る
- [ ] 魔物登場時点で味方パラメータ欄が出る
- [ ] 魔物登場時点で敵スプライトが出る
- [ ] 初回コマンド時に画面がズレない
- [ ] カーソル移動で文字がちらつかない
- [ ] 敵攻撃前後に長い白画面が出ない
- [ ] 戦闘後にフィールドへ戻れる

## 注意点

- 今回のBGMは既存曲のコピーではなく、SaGa風の方向性を意識したオリジナル短ループ。
- hUGEdriverは未導入。
- 音量が大きい場合は `audio.c` の `NR50_REG` を調整する。
