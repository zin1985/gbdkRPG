# rpg120 sample frame + item/equip ui

## 反映内容
- ユーザー提示の二重線ウインドウ枠イメージを共通UI枠として採用。
- 会話ウインドウ、メニュー囲み、戦闘メッセージ枠、戦闘コマンド枠を同系統で統一。
- `jp_draw_bkg_frame()` を追加し、Bank 7 のメニュー系からも同じ枠を利用可能にした。

## 実装内容
### もちものUI
- フレーム付き画面へ更新。
- Aで使用対象選択ポップアップを表示。
- `やくそう / ポーション / まほう草 / どくけし` をフィールド使用可能に実装。
- 使用時に在庫を1消費。

### そうびUI
- フレーム付き画面へ更新。
- 武器 / 防具 / 装飾 を上下で選択。
- Aで候補ポップアップを表示。
- 所持品にある対応装備を選んで装備可能。
- 旧装備は所持品へ戻し、新装備は所持品から1減らす。
- 左右で仲間切替。

## 注意
- この環境ではビルド未実施。
- 既存の装備初期配布仕様上、初期装備と所持品の重複があり得る。
