# rpg087 growth rank design

## 方針

この版では、スレッドで検討した成長システム拡張のうち、既存の戦闘UIを壊しにくい土台だけを実装する。

- HP/MP上限: 999
- その他能力上限: 255
- 成長結果は GrowthResult に保存する
- 敵ランク battle_enemy_rank を戦闘開始時に保存する
- 成長発生率は rank 1〜5 のテーブルで管理する
- 上限到達時は別能力へ流れ込ませず、成長なしにする

## 今回入れた対象能力

- HP
- MP
- ちから
- まもり
- とくぎ
- かいふく
- すばやさ

## 今回まだ入れないもの

- 複数キャラ成長
- 複数成長ログ
- 技ひらめき
- 武器熟練度
- メッセージキュー
- MP消費制
