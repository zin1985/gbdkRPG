# UI Frame Adopted Standard (rpg119)

## 採用内容
- 直近サンプルとして確認した「角 + 線」3階調枠を正式採用。
- メッセージウインドウとメニュー囲みを同一フレームスプライトで統一。
- 色は Game Boy の 4色中、背景色(白) + 3階調を使用。

## 実装箇所
- `jpfont.c`
  - `jp_frame_tiles[]` に共通UI枠タイルを保持
  - `jp_window_prepare()` で会話ウインドウを描画
- `main.c`
  - `draw_bkg_box()` で戦闘枠・メニュー枠・各種小枠を描画

## 統合対象
- 会話ウインドウ
- 戦闘メッセージ枠
- 戦闘コマンド枠
- メニュー小枠
- パラメータ表示系の囲み枠

## 方針
- 交差点専用タイルは作らない
- 角4種 + 辺4方向 + 白地で構成
- 枠の重なりはそのまま重ねて見せる（SaGa / GB風の見せ方）

## 備考
- 既存の `draw_bkg_box()` がメニュー系も含めて同じタイル群を使うため、UI統一済み。
