# rpg138 デバッグ用ノーエンカウントアクセサリー設計

## 概要
デバッグ中にダンジョンや古代遺跡のNPC・宝箱・罠床・マップ遷移を確認しやすくするため、装備中だけランダムエンカウントを止める装飾品を追加した。

## アイテム
- ID: `ITEM_DEBUG_NO_ENCOUNT`
- 名称: `退魔のすず`
- 種別: 装飾品
- 効果: アクティブメンバーの誰かが装備している間、`field_feature_encounter_rate()` が0を返す。

## Bank 0保護
`main.c` のランダムエンカウント判定本体には処理を追加せず、既存の `field_feature_encounter_rate()` 呼び出し先で判定する。
これにより、Bank 0側の分岐増加を避ける。

## 注意
完全なイベント戦闘や固定敵接触を止めるものではなく、通常の歩行ランダムエンカウント停止用。
