# rpg137 dungeon NPC placement

## 目的
洞窟ダンジョンと古代遺跡に、エリアごとのNPC配置を追加する。

## 方針
- Bank 0にNPC配置テーブルを置かない。
- `actor_runtime_apply_area_npcs(area)` を `actor_runtime.c` の banked 側に追加。
- actor[0] / actor[2] をNPC枠として使い、actor[1] は従来の敵テンプレートとして維持。
- フィールドではNPCを非表示。町・港町・洞窟・遺跡でエリア別配置へ切替。

## 配置
- 洞窟: 案内NPC `(4,5)`、負傷した探索者 `(10,11)`
- 遺跡: 学者NPC `(2,4)`、石の声を聞くNPC `(10,12)`

## 注意
- NPC本体は2体までに抑え、OAM使用量を安定化。
- ダンジョンNPCとの会話で、敵復活プロンプトが誤表示されないように、復活プロンプトは町系エリア限定にした。
