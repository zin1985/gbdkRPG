# CHECK rpg087_growth_rank

## 目的

rpg086_window_reset を基準に、スレッドで検討した成長システム拡張のうち、すぐ実装可能な土台を追加する。

## 実装確認

- [ ] HP/MPの上限が 999 になっている
- [ ] ちから/まもり/とくぎ/かいふく/すばやさの上限が 255 になっている
- [ ] 主人公に MP ステータスが追加されている
- [ ] 戦闘画面の主人公 M 表示が固定値ではなく現在MPを表示している
- [ ] 勝利時の成長候補に MP / とくぎ / かいふく が含まれている
- [ ] 選ばれた能力が上限の場合、すばやさへ流れ込まず成長なしになる
- [ ] 敵ランク battle_enemy_rank が戦闘開始時に保存される
- [ ] 成長発生率が敵ランクテーブルで決まる
- [ ] GrowthResult に actor_id / stat_type / old_value / new_value が保存される

## VBA確認

- [ ] 起動して白画面にならない
- [ ] フィールド表示が崩れない
- [ ] STARTメニューの「つよさ」で HP / MP が表示される
- [ ] ランダムエンカウントする
- [ ] 戦闘突入時に主人公のH/M表示が出る
- [ ] 戦闘勝利後に成長メッセージが出る場合がある
- [ ] HPが上がった場合に次回以降の最大HPへ反映される
- [ ] MPが上がった場合に「つよさ」へ反映される
- [ ] 戦闘後にフィールドへ戻れる
- [ ] 戦闘UI位置が rpg086_window_reset から大きく崩れていない
