# rpg200_shop_clerk_villager_icon_block

## 対応内容
- 街の店内にいる店員スプライトを、通常の村人/NPCスプライトへ変更。
  - `actor_runtime.c`
  - 町エリアの店員を上部店舗内の `(4,2) / (8,2) / (12,2)` に配置。
  - `actor_runtime_setup_npc()` を使うため、村人/NPCタイルを使う。
- 店員の足元/店内タイル自体もイベント化。
  - `field_feature_runtime.c`
  - 店員タイル `(4,2) / (8,2) / (12,2)` と、前/左右ブロックから各店を開ける。
- 店アイコンとブロックの重ね表示を、BG合成タイルで実装。
  - GBのBGは透明レイヤーを重ねられないため、事前合成した「ブロック＋アイコン」タイルに変更。
  - 道具アイコン看板、剣アイコン看板を16x16の合成タイルにした。
  - `MAP_GFX_TILE_COUNT` は増やしていない。
  - 町表示時だけ既存スロットへ上書きする方式を維持。

## 容量について
- BG合成方式では、1種類の16x16看板 = 4タイル = 64 bytes。
- 道具/剣の2種類で合計128 bytes程度のROM増。
- `MAP_GFX_TILE_COUNT` を増やしていないため、VRAM側のタイル数は増やしていない。
- OBJで重ねる方式も可能だが、OAM数/scanline制限/ちらつきのリスクがあるため、店看板はBG合成の方が安全。
